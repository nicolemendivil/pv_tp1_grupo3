++++++++Desripcion html primera parte++++++
En esta primera parte del trabajo desarrollamos la estructura de un sitio web sobre gestión de proyectos educativos utilizando HTML.
Creéamos las páginas index.html, proyectos.html, detalle.html y perfil.html, aplicando etiquetas semánticas como header, nav, main, section y footer para organizar el contenido.
También vinculamos las páginas entre sí mediante enlaces para permitir la navegación dentro del sitio. El objetivo fue lograr una estructura clara y ordenada sin usar estilos.

++++++++Descripcion css segunda parte+++++++
En esta parte del trabajo aplicamos estilos a las páginas HTML utilizando CSS para mejorar la apariencia del sitio.
Usamos un archivo externo (styles.css) para organizar los estilos, además de variables en :root para mantener los mismos colores y tipografía en todas las páginas.
También utilisamos Flexbox para el menú y los proyectos, y Grid para organizar el perfil. Agregamos algunos efectos como hover en botones y enlaces para darle un poco de interactividad.
El objetivo fue que todas las páginas tengan un diseño claro, ordenado y consistente.
